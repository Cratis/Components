# Dialogs

The Dialogs module provides common dialog components for user interactions.

## Components

- **Dialog**: Base dialog component for creating custom dialogs
- **BusyIndicatorDialog**: Dialog with loading spinner
- **ConfirmationDialog**: Dialog for confirming user actions
